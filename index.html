常見問題

如何申請 API 金鑰？
請前往 platform.openai.com/api-keys，登入或註冊後即可查看或建立新的 API 金鑰。

API 金鑰僅在申請成功時顯示一次，請務必妥善保存，如遺失需重新申請。

切勿將金鑰公開或托管於公開程式庫，以免造成資安風險。如有遺失或疑似外洩，請立即於 OpenAI 平台上刪除並重建新金鑰。

提醒：申請完記得要儲錢進去喔！
這個 App 一個月要花多少錢？
如果你一個月呼叫 500 次「自然語言記帳」，再加上「分析」和「報告」各一次，大約只要美金 1 塊左右，也就是新台幣 30 元出頭。

還可以透過批次呼叫（一句話講多一點商品）省更多，最多可壓到不到新台幣 20 元！
如何使用自然語言記帳？
在輸入框中輸入「今天午餐花 150 元」，App 即可自動解析並記錄。

你也可以說：「昨天買咖啡 65 元、晚餐便當 85 元」這樣一次記多筆！
為何無法解析句子？
請確認句子包含完整的「品項」和「金額」資訊。

若仍無法解析，可以嘗試長按「+」按鈕手動新增記錄。
如何匯出 CSV？
在「設定」頁面點選「匯出 CSV 檔案」，即可下載完整的記帳資料。
如何設定訂閱自動扣款？
進入「設定 → 訂閱」新增訂閱名稱及金額，App 會自動每月新增訂閱支出記錄。
